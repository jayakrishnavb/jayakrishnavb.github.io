Course Title

Design and Analysis of Algorithms (24CST2225)

Course Description

This course introduces algorithm design techniques, performance analysis, complexity analysis, and application of algorithms to solve computational problems. Learners will understand efficiency trade-offs, apply design paradigms, analyze time-space complexity, and evaluate algorithmic strategies.

Course Objectives

  • Understand fundamental algorithmic concepts and asymptotic notations.
  • Learn various algorithm design techniques including greedy, divide-and-conquer, and DP.
  • Analyze time and space complexity of algorithms.
  • Apply algorithmic approaches to practical computational problems.
  • Understand NP-Completeness and related complexity classes.

Module 1 – Introduction to Algorithms

  • Algorithm – definition and characteristics
  • Performance analysis and metrics
  • Asymptotic notations: Big-O, Ω, Θ
  • Recurrence relations and solution methods
  • Master Theorem (basic cases)

Module 2 – Divide and Conquer

  • Concept and paradigm
  • Binary search
  • Merge sort and Quick sort
  • Selection algorithms
  • Strassen’s matrix multiplication

Module 3 – Greedy Method

  • Greedy-choice property and optimal substructure
  • Knapsack problem (fractional)
  • Job sequencing with deadlines
  • Minimum cost spanning tree – Kruskal, Prim
  • Dijkstra’s single-source shortest path

Module 4 – Dynamic Programming

  • Optimal substructure
  • 0/1 Knapsack problem
  • Matrix-chain multiplication
  • Floyd-Warshall algorithm
  • Longest common subsequence

Module 5 – Graph Algorithms

  • Breadth-first search (BFS)
  • Depth-first search (DFS)
  • Topological sorting
  • Graph traversal applications

Module 6 – Complexity Theory

  • P, NP, NP-Complete, NP-Hard classes
  • Reductions
  • Basic NP-Complete problems

Textbooks

  • Cormen, Leiserson, Rivest, Stein – Introduction to Algorithms
  • Kleinberg & Tardos – Algorithm Design

References

  • Horowitz & Sahni – Fundamentals of Computer Algorithms
  • Aho, Hopcroft, Ullman – Design and Analysis of Computer Algorithms
